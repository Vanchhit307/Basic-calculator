# 🧮 Basic Calculator - Java Swing

A simple, user-friendly calculator application built with **Java Swing**. This project demonstrates fundamental GUI programming concepts in Java.


## ✨ Features

- ✅ **Basic Arithmetic Operations** — Addition, Subtraction, Multiplication, Division
- ✅ **Decimal Point Support** — Handle floating-point numbers
- ✅ **Clear (CLR)** — Reset everything with one click
- ✅ **Delete (DEL)** — Backspace functionality to remove last digit
- ✅ **Error Handling** — Prevents division by zero with error message
- ✅ **Clean UI** — Color-coded buttons for better user experience
- ✅ **Responsive Layout** — Grid-based button arrangement
